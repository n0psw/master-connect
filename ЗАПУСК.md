# 🚀 Инструкция по запуску MasterConnect

## ⚡ Быстрый запуск (5 минут)

### 1️⃣ Backend (Терминал 1)

```bash
cd apps/api

# Активировать виртуальное окружение
venv\Scripts\activate

# Запустить сервер
python -m uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
```

✅ **Backend запущен**: http://localhost:8000  
📚 **API Docs**: http://localhost:8000/docs

---

### 2️⃣ Frontend (Терминал 2)

```bash
cd apps/web

# Запустить dev сервер
npm run dev
```

✅ **Frontend запущен**: http://localhost:3000

---

## 🎯 Что дальше?

### **Создать тестовых пользователей:**

1. Откройте http://localhost:3000
2. Нажмите "Регистрация"
3. Создайте студента и ментора

### **Протестировать основные функции:**

#### Студент:
1. Войдите как студент
2. Заполните профиль
3. Перейдите в каталог менторов
4. Забронируйте консультацию
5. Загрузите доказательство оплаты

#### Ментор:
1. Войдите как ментор
2. Заполните профиль
3. Настройте расписание
4. Просмотрите бронирования

#### Админ:
1. Создайте пользователя и измените роль в БД:
   ```sql
   UPDATE users SET role = 'ADMIN' WHERE email = 'admin@test.com';
   ```
2. Войдите как админ
3. Проверьте оплату студента
4. Верифицируйте ментора

## ⚠️ Частые проблемы

### Backend не запускается
```bash
# Проверить виртуальное окружение
venv\Scripts\activate

# Переустановить зависимости
pip install -r requirements.txt
```

### Frontend не подключается к backend
```bash
# Проверить .env файл
cat apps/web/.env
# Должно быть: VITE_API_URL=http://localhost:8000/api/v1

# Проверить что backend запущен
curl http://localhost:8000/health
```

### CORS ошибки
Добавьте в `apps/api/.env`:
```env
BACKEND_CORS_ORIGINS=["http://localhost:3000","http://localhost:5173"]
```
